# Кластеризация датасета MNIST с помощью K-Means

Проект посвящен исследованию структуры изображений рукописных цифр из датасета MNIST методами обучения без учителя (*Unsupervised Learning*). В работе проводится визуальный анализ данных с помощью снижения размерности, подбор оптимального числа кластеров и интерпретация полученных результатов.

## Стек технологий
* **Python 3**
* **Scikit-learn** (KMeans, PCA, TSNE)
* **Matplotlib** (Визуализация результатов)
* **NumPy**

## Основные этапы проекта

1. **Разведочный анализ и снижение размерности:**
   * Применение **PCA** для линейного снижения размерности до 2 компонент.
   * Применение **t-SNE** для сохранения локальных структур данных на плоскости. t-SNE показал значительно более качественное разделение на выраженные кластеры-облака.

2. **Обучение модели и подбор гиперпараметров:**
   * Построен цикл обучения `K-Means` для $k \in [2, 15]$.
   * Для каждого шага рассчитаны метрики **Инерции (Inertia)** и **Силуэтной меры (Silhouette Score)**.
   * На основе комбинированного графика метрик выбрано оптимальное число кластеров $k = 10$.

3. **Анализ центроидов:**
   * Проведена визуализация усредненных изображений (центроидов) для каждого из 10 кластеров.
   * Выявлен инсайт: алгоритм выделил под цифры "1" и "0" по два отдельных кластера из-за разницы в стиле написания (наклон, толщина). Из-за этого цифры "4" и "5" смешались с геометрически похожими кластерами.
